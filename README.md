# 🎨 Генератор Логотипов с ИИ

Веб-приложение и Telegram-бот для генерации уникальных логотипов с помощью нейросети **Yandex Art API**.

## ✨ Возможности

- 🌐 **Веб-интерфейс**: Удобная форма для создания логотипов в браузере
- 🤖 **Telegram-бот**: Генерация логотипов прямо в мессенджере
- 🎨 **8 стилей**: От минимализма до винтажа
- 🔧 **Доработка**: Возможность уточнить детали после генерации
- 💾 **Скачивание**: Сохранение результата в высоком качестве
