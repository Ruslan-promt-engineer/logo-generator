# 🎨 Генератор Логотипов с ИИ

> Веб-приложение и Telegram-бот для генерации уникальных логотипов с помощью нейросети **Yandex Art API**.

![aiogram](https://img.shields.io/badge/aiogram-3.x-blue.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🌐 **Веб-интерфейс** | Удобная форма для создания логотипов в браузере |
| 🤖 **Telegram-бот** | Генерация логотипов прямо в мессенджере |
| 🎨 **8 стилей** | Минимализм, модерн, геометрия, винтаж, техно, корпоратив, креатив, элегант |
| 🔧 **Доработка** | Возможность уточнить детали после генерации (с сохранением структуры) |
| 💾 **Скачивание** | Сохранение результата в высоком качестве (PNG) |
| 🔗 **Синхронизация** | Единая логика генерации для веба и бота |

## 🚀 Быстрый старт

### 1️⃣ Клонирование репозитория
```bash
git clone https://github.com/Ruslan-promt-engineer/logo-generator.git
cd logo-generator
```
### 2️⃣ Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3️⃣ Настройка переменных окружения
Создайте файл .env в корне проекта по образцу .env.example:
```bash
# ─────────────────────────────────────────
# 🔑 Yandex Cloud API
# ─────────────────────────────────────────
YANDEX_API_KEY=ваш_ключ_от_сервисного_аккаунта
FOLDER_ID=ваш_id_каталога_(начинается_на_b1g)

# ─────────────────────────────────────────
# 🤖 Telegram Bot
# ─────────────────────────────────────────
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather

# ─────────────────────────────────────────
# ⚙️ Flask Settings
# ─────────────────────────────────────────
FLASK_ENV=development
FLASK_DEBUG=1
```
⚠️ Важно: Никогда не загружайте файл .env в репозиторий! Он уже добавлен в .gitignore.

### 4️⃣ Запуск
🌐 Веб-приложение:
```bash
python app.py
```
Откройте в браузере: 👉 http://localhost:5000

### 🤖 Telegram-бот:

```bash
python telegram_bot.py
```
Найдите бота в Telegram по @username и нажмите /start

### 🔧 Доступные маршруты (веб)

### ⚠️ Важные замечания
🔐 Безопасность
Никогда не коммитьте файл .env — он содержит секретные ключи
Регулярно обновляйте API-ключи и токены
Используйте FLASK_DEBUG=0 в продакшене

🌐 Деплой
Для продакшена рекомендуется использовать Gunicorn + Nginx
Настройте HTTPS для безопасной передачи данных
Используйте RotatingFileHandler для логирования

🎨 Генерация
Качество логотипа зависит от детализации промпта
Используйте пресеты стилей для предсказуемого результата
Один seed не гарантирует идентичную композицию при изменении промпта
